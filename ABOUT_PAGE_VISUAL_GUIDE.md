## 🎨 Color Legend

### Section Backgrounds
- **White cards:** `bg-white dark:bg-gray-800`
- **Gradient heroes:** Teal/Cyan/Purple variations
- **Accent boxes:** Light colored with borders

### Interactive States
- **Default:** Base color
- **Hover:** Slightly darker shade + scale/shadow
- **Active nav:** Teal background with bold text
- **Focus:** Outline ring visible

---

## 🔄 Interactive Elements Map

```
SCROLLABLE AREAS:
├─ Main page (smooth scroll)
├─ Sidebar links (jump to sections)
└─ Scroll-to-top button (return to hero)

CLICKABLE ELEMENTS:
├─ Hero CTA → Scrolls to #mission
├─ Sidebar links → Jump to sections (8 total)
├─ Quick actions → Navigate to pages (2 buttons)
├─ Team member link → /discussions/members
├─ Partner CTA → /contact
├─ Support cards → 4 different destinations
└─ Final CTA → /contact

HOVER EFFECTS:
├─ All buttons (color change + shadow)
├─ Cards (scale-105 + shadow enhancement)
├─ Links (color change)
└─ Sidebar items (background color)
```

## 📏 Spacing System

```
Hero Section:       py-20 md:py-28  (80px → 112px)
Main sections:      py-12           (48px)
Section gaps:       space-y-16      (64px between sections)
Card padding:       p-6 or p-8      (24px or 32px)
Grid gaps:          gap-6 or gap-8  (24px or 32px)
```

---

## 🎯 Visual Hierarchy

```
LEVEL 1 (Largest):
├─ Page title (Hero)
└─ Section headings (h2)

LEVEL 2:
├─ Subsection headings (h3)
└─ Card titles

LEVEL 3:
├─ Body text
└─ Descriptions

LEVEL 4 (Smallest):
├─ Meta information
└─ Captions
```

---

## 🖼️ Image Placement

```
Hero Section:         Background image (full-width)
Mission Section:      Left side image (or icon fallback)
Story Section:        Floating right image
Homelands Section:    Map placeholder (full-width)
Team Section:         Circular avatars (gradient fallback)
```

---

## 📊 Grid Layouts

```
DESKTOP (lg):
├─ Mission: 2 columns (image + text)
├─ Approach: 2×2 grid (4 cards)
├─ Team: 2 columns (4 members)
├─ Impact: 4 columns (statistics)
└─ Support: 4 columns (CTAs)

TABLET (md):
├─ Most 2-column grids maintained
└─ Some collapse to single column

MOBILE (sm):
└─ All single column
```

---

## 🎭 Animation Details

```
SMOOTH SCROLL:
- Duration: CSS-based (instant but smooth)
- Offset: 80px (accounts for navbar)

HOVER ANIMATIONS:
- Transform: scale-105 (5% larger)
- Transition: all 200ms

SCROLL BUTTON:
- Appears: After 400px scroll
- Animation: Fade in
- Position: Fixed bottom-right
```

## 📐 Sidebar Behavior

```
DESKTOP (lg+):
├─ Width: 25% (1/4)
├─ Position: sticky
├─ Top offset: 96px (6rem)
└─ Stays visible while scrolling

MOBILE:
├─ Width: 100%
├─ Position: static
├─ Stacks above main content
└─ Scrolls with page
```
